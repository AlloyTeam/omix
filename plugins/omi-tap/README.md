## omi-tap

Support tap event in your [Omi](https://github.com/AlloyTeam/omix) project.

---

## Install

``` js
npm install omi-tap
```

## Usage

```js
import 'omi-tap';

class App extends Component {
    tapHandler(evt) {
        alert('Hello Omi + Parcel!')
    }

    render() {
        return (
            <div>
                <h1 omi-tap tap={this.tapHandler}>Hello Omi + Parcel 📦 🚀</h1>
            </div>
        )
    }
}

Omi.render(new App(),"#container");
```
